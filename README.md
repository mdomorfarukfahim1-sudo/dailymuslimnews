# 🛍️ Daily Muslim News Shop

An Islamic online shop featuring products like dates, attar, prayer mats, clothing, and more.

---

## 📌 Overview

**Daily Muslim News Shop** is a simple and user-friendly e-commerce website where users can browse and view Islamic lifestyle products.

---

## 🧾 Products Available

* 🧺 Ajwa Dates
* 🌴 Madina Dates
* 🌙 Mabroom Dates
* 🕌 Prayer Mat Set
* 🌿 Attar (Perfume)
* 🎁 Premium Attar Gift Box
* 🧕 Stylish Burqa
* 👕 Panjabi

---

* 🛒 Product listing with images and prices
